## Human Verification

| AC                           | Description                                                                             | Why Not Automated                                                                                                                                                                                                                                                                                                                                               | Verification Approach                                                                                                                                                                                                                    |
| ---------------------------- | --------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| p3-u08-discover-wiring.AC3.1 | `sync.events.on("sync:complete", ...)` is subscribed before `server.connect(transport)` | This is a source-level ordering constraint in `src/index.ts`. The subscription happens inside `setupDiscoverFeature()` which is `await`-ed, and `server.connect(transport)` is called afterwards. Testing call ordering across the entry point would require an integration test of the full `main()` function with real stdio transport, which is impractical. | Code review: verify in `src/index.ts` that the `await setupDiscoverFeature(...)` call appears before the `await server.connect(new StdioServerTransport())` call. Run `git diff src/index.ts` after implementation and confirm ordering. |
| p3-u08-discover-wiring.AC5.1 | `src/paprika/sync.ts` is not modified by this implementation                            | This is a file-level constraint that cannot be expressed as a runtime test. It is a property of the implementation diff, not of program behavior.                                                                                                                                                                                                               | Run `git diff src/paprika/sync.ts` and verify no output. This check is included in the implementation plan's verification steps (Tasks 5 and 7).                                                                                         |
