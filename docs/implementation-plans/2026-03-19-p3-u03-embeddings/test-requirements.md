# Test Requirements: p3-u03-embeddings

This document maps every acceptance criterion from the p3-u03-embeddings design plan to a specific automated test or documented human verification step.

All tests are colocated in a single file using MSW for HTTP mocking and vitest as the runner.

---

## Automated Test Requirements

| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------- | --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |

| p3-u03-embeddings.AC3.2 | 401 response throws `EmbeddingAPIError` (permanent, no retry)                                                                                                | unit      | `src/features/embeddings.test.ts` | MSW handler returns 401; call `embedBatch`; assert `EmbeddingAPIError` is thrown with `status: 401`; verify handler was called exactly once                                                                                                                              |

| p3-u03-embeddings.AC5.4 | Empty category array produces no "Categories:" line                                                                                                          | unit      | `src/features/embeddings.test.ts` | Create recipe with populated fields but pass `[]` as category names; call `recipeToEmbeddingText`; assert output does not contain "Categories:"                                                                                                                          |
