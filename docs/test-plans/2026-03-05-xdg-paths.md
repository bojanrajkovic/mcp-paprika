| 3.2  | Run the same for `getCacheDir`                                                                                   | Prints a path like `/home/<user>/.cache/mcp-paprika`                           |

| 3.4  | Run the same for `getLogDir`                                                                                     | Prints a path like `/home/<user>/.local/state/mcp-paprika`                     |

| 3.6  | Set `XDG_CONFIG_HOME=/custom/config`, then run `getConfigDir()`                                                  | Returns `/custom/config/mcp-paprika` (env-paths respects XDG environment vars) |
